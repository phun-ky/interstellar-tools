# Variable: J2000_UTC

```ts
const J2000_UTC: Date;
```

Defined in:
[date.ts:36](https://github.com/phun-ky/interstellar-tools/blob/de093dcb047b9aaae6ea3706fd8ef5ec2dd7b0ae/packages/constants/src/date.ts#L36)

**J2000 epoch as a UTC timestamp** for JavaScript usage.

J2000 is _defined_ as **JD
[`2451545`](https://github.com/phun-ky/interstellar-tools/commit/2451545).0
(TT)**, i.e. 2000-01-01 12:00:00 **TT**. At that epoch **TT − UTC = 64.184 s**,
so the corresponding UTC instant is: **2000-01-01T11:58:55.816Z**.

## Example

```ts
// Milliseconds from UTC midnight to J2000 (UTC):
const deltaMs = +J2000_UTC - +EPOCH_2000_UTC_MIDNIGHT; // 43_135_816 ms

// If you need the TT-based Julian centuries since J2000:
//   T = (JD_TT - J2000_TT) / 36525
```
